Weather

Website with weather information

  • Url: carlos5leal.github.io/Weather/
  • Languages: HTML, CSS and JavaScript
  • Responsive: Bootstrap
  • JavaScript: jQuery, JSON and Ajax
  • API's: Accuweather, Mapbox, ipgeolocation and Unsplash

Website that indicates the weather forecast (current, next days, next hours and last hours) based on the user's coordinates and that also allows searching from other cities. An API is also used to search for an image of the city.
